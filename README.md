# JaegerOps Portfolio Website

A simple, clean cybersecurity portfolio website for **JaegerOps** built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` – page structure and content sections
- `style.css` – dark, professional styling and responsive layout
- `script.js` – mobile menu toggle + dynamic footer year

## Included Sections

- Hero
- About
- Projects
- Writeups
- Contact

## Projects Highlighted

- JaegerOps Website
- Tesla Recon Project
- Hack The Box labs

## How to Use

1. Open `index.html` in your browser.
2. Edit text in `index.html` to update your profile, projects, and contact info.
3. Adjust colors/spacing in `style.css` as needed.
4. Extend interactions in `script.js` if you want more behavior.

## Customization Tips

- Replace `youremail@example.com` with your real email.
- Update LinkedIn and GitHub URLs in the Contact section.
- Duplicate project cards in the Projects section to add more entries.
- Keep the same section IDs so navigation links continue to work.
